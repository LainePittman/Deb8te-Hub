Deb8teHub Project Setup Steps:

  • Create new directory to house Django project
  • Download Deb8teHub django project configuation directory from https://github.com/LainePittman/Deb8te-Hub.git into created directory
  • Create venv inside of created directory using python -m venv venv_name
  • Launch into venv you created using ./venv_name/bin/activate
    • May need to use source command if working in Linux
    • Check folder structure of venv_name directory if activate not found
  • Install Django package to use runserver/migrate commands using python -m pip install Django
  • Cd into Deb8teHub project directory and run python manage.py runserver
  • Navigate to preferred browser and open http://127.0.0.1:8000/ to view app
  • Use http://127.0.0.1:8000/admin for admin view of site to check db transactions
